Image Captioning is a task where a caption is generated for the image based on its content. For instance, if an image contains a "Cat and Dog" an example caption is "A Cat and Dog are fighting". Here, we try to explain why the following caption is generated given the image. We provide explanations via heatmaps. In the Heatmaps, red Color signifies more important regions and Blue Color signifies regions with no importance.
